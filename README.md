# Mirza Eka

## Install Dependencies
```
npm install
```

## Setup Database
Open index.js and change Database Configuration
```
  host: 'localhost',
  user: 'CHANGE_WITH_YOUR_USER',
  password: 'CHANGE_WITH_YOUR_PASS',
  database: 'CHANGE_WITH_YOUR_DB',
```

## Run Server
```
node index.js
```

## Create Tables

```
http://localhost:3000/api/createtable
```

### Run The Apps

```
http://localhost:3000/
```
